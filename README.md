# xApp Builder [![xapp-builder](https://snapcraft.io/xapp-builder/badge.svg)](https://snapcraft.io/xapp-builder)

Application for Windows / Mac OSX / Linux SNAP to test & debug xApps locally.

1. npm install
2. npm run watch
